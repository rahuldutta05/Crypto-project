to run the code
1. run in command prompt
    git clone https://github.com/rahuldutta05/Crypto-project

2. for backend,
    cd backend
    pip install -r requirements.txt
    python app.py
    backend should be running at http://127.0.0.1:5000/

3. for frontend,
    cd frontend 
    npm install 
    npm install axios
    npm run dev
    frontend should be running at http://localhost:5174/
